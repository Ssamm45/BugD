Manage the current database
	init <filename>       Creates a new database named <filename>
	set <filename>        Sets the database to <filename>
	current               Prints out the path of the current database
	list                  Lists all of the entries in the current database

Edit entries in the database
	new                   Creates a new entry in the current db
	view <entry-id>       View the given entry
	edit <entry-id>       Edit the given entry
	delete <entry-id>     Deletes the given entry

Misc.
	help                  View usage information
